The CSS margin properties are used to create space around elements, outside of any defined borders. margin-top margin-right margin-bottom margin-left p { margin: 25px 50px 75px 100px; } All the margin properties can have the following values: auto - the browser calculates the margin length - specifies a margin in px, pt, cm, etc. % - specifies a margin in % of the width of the containing element inherit - specifies that the margin should be inherited from the parent element Tip: Negative values are allowed. div { border: 1px solid red; margin-left: 100px; } p.ex1 { margin-left: inherit; (margin left will be 100px) }
